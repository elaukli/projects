This is a tool for parsing the data from a CU Boulder FCQ result page for a teacher or class. To see how it works you can use the provided .CSV files or copy->paste from any FCQ page into a new excel sheet with matching formatting to the examples

The menu is pretty self explanatory

compile with: g++ -std=c++11 fcq.cpp fcqDriver.cpp
run with 2 files: ./a.out byteacher.csv byclass.csv
